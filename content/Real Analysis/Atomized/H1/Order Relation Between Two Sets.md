# Order Relation Between Two Sets

*Original Note: [[L04]]*

> [!thm] Comparing sup A and inf B
> If A, B ⊆ ℝ are nonempty and for all a ∈ A and b ∈ B we have a ≤ b, then
> $$\sup A \le \inf B.$$
> 
> > [!pf] Proof
> > Since a ≤ b for all a ∈ A and b ∈ B, each a ∈ A is a lower bound of B. Hence a ≤ inf B for all a ∈ A, so inf B is an upper bound of A. Therefore sup A ≤ inf B.
