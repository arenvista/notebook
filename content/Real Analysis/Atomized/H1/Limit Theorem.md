# Limit Theorem

*Original Note: [[L07]]*

> [!def] Definition (Limit of a Sequence)
> A sequence $(x_n)$ converges to $x$ if
> $$\forall \epsilon > 0, \ \exists N(\epsilon) \in \mathbb{N} \text{ such that } \forall n \ge N(\epsilon), \ |x_n - x| < \epsilon.$$
> We write $x_n \to x$ as $n \to \infty$.
